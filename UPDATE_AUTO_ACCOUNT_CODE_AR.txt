West ERP 1.9.3 — الكود التلقائي للحسابات

تمت إضافة:
- اقتراح كود الحساب الفرعي تلقائيًا عند اختيار الحساب الأب.
- النظام يراجع الحسابات الفرعية الموجودة ويقترح الرقم التالي.
- أمثلة:
  تحت 1000: 1100 ثم 1200 ثم 1300.
  تحت 1100: 1110 ثم 1120 ثم 1130.
  تحت 1110: 1111 ثم 1112 ثم 1113.
- يمكن تعديل الكود المقترح يدويًا قبل الحفظ.
- دعم أكواد غير رقمية مثل ABC-01 وABC-02.

طريقة التحديث:
1. ارفع app.py و chart_of_accounts.html إلى GitHub.
2. Commit changes.
3. Render > Manual Deploy > Deploy latest commit.
4. بعد Live حدّث الصفحة بـ Ctrl+Shift+R.
