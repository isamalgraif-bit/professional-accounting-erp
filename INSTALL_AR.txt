طريقة التحديث إلى West ERP Professional 1.1.0

1. فك ضغط الملف.
2. افتح مستودع GitHub professional-accounting-erp.
3. اختر Add file ثم Upload files.
4. ارفع جميع الملفات داخل المجلد بعد فك الضغط.
5. وافق على استبدال الملفات القديمة.
6. اضغط Commit changes.
7. افتح Render.
8. اختر Manual Deploy ثم Deploy latest commit.
9. انتظر حتى تصبح الحالة Live.
10. افتح البرنامج وسجل الدخول.
11. غيّر كلمة المرور فورًا.
12. افتح الإعدادات وأدخل بيانات الشركة وارفع الشعار.
