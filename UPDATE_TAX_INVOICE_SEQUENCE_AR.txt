تحديث الفاتورة الضريبية والترقيم المتسلسل

تمت إضافة:
- فاتورة ضريبية / TAX INVOICE.
- رقم فاتورة متسلسل تلقائي حسب السنة.
- مثال: INV-2026-000001 ثم INV-2026-000002.
- يبدأ تسلسل جديد عند بداية سنة جديدة.
- الفواتير القديمة تحتفظ بأرقامها.
- الترقيم الجديد يطبق على الفواتير الجديدة.
- يستمر QR وUUID والطباعة وPDF والتقارير.

طريقة التركيب:
1. فك الضغط.
2. في GitHub اختر Add file ثم Upload files.
3. ارفع الملفات الأربعة ووافق على الاستبدال.
4. اضغط Commit changes.
5. في Render اختر Manual Deploy ثم Deploy latest commit.
6. انتظر حتى تصبح الحالة Live.
