# نظام ERP محاسبي مبسط – النسخة الأولى

هذه نسخة أولية تعمل فعليًا، باللغة العربية، ومناسبة كبداية لشركة متعددة الفروع.

## الوظائف الموجودة
- تسجيل دخول.
- لوحة مؤشرات.
- إعدادات اسم الشركة بالعربية والإنجليزية.
- الفروع.
- العملاء.
- الموردون.
- فواتير مبيعات مع احتساب ضريبة 15%.
- المصروفات.
- المخزون.
- الموظفون والرواتب الأساسية.

## التشغيل على ويندوز

1. ثبّت Python 3.11 أو أحدث.
2. افتح مجلد المشروع في Command Prompt.
3. نفّذ:

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

4. افتح المتصفح:
`http://127.0.0.1:5000`

## بيانات الدخول
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

غيّر كلمة المرور والمفتاح السري قبل الاستخدام الفعلي.

## ملاحظات مهمة
هذه نسخة MVP وليست بعد نظامًا معتمدًا نهائيًا لدى هيئة الزكاة. الربط الفعلي مع ZATCA يحتاج:
- XML وفق UBL 2.1
- UUID وInvoice Hash
- QR TLV
- CSID وشهادات التوقيع
- Clearance وReporting APIs
- سجل تدقيق ومنع تعديل الفواتير بعد الاعتماد

يمكن تطوير هذه النسخة لتشمل ذلك دون إعادة بناء النظام من البداية.
